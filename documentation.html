Introduction

Windfield is an intelligent operations platform that connects property data, market intelligence, and automated workflows into a unified command center for real estate professionals.

What is Windfield?

Windfield transforms how real estate teams operate by combining agentic AI, structured data pipelines, and the MCP protocol into a single platform. Instead of toggling between disconnected tools, you get a centralized system that understands your properties, your market, and your brand.

The platform is built around three pillars: property intelligence that enriches and scores every asset in your portfolio, market scanning that surfaces opportunities before your competitors see them, and content automation that turns raw data into polished marketing materials.

Windfield uses the Model Context Protocol (MCP) to connect AI agents with your data sources. All API calls are authenticated, rate-limited, and logged for full audit traceability.

Key Features

The platform ships with a complete suite of tools designed for real estate operations at scale.

Property Intelligence

Automated enrichment, scoring, and profile generation for every property in your portfolio.

Market Scanner

Real-time market signals, competitor tracking, and opportunity detection across your target areas.

Brand Compass

Voice fingerprinting and content guardrails that keep every asset on-brand, automatically.

Content Studio

Generate articles, social posts, and listing descriptions from structured property data.

Quick Start

Get up and running with Windfield in three steps.

1

Install the SDK

Add the Windfield SDK to your project using npm or yarn. The package includes TypeScript types out of the box.

2

Configure your API key

Create an API key from the Windfield dashboard and set it as an environment variable in your project.

3

Make your first request

Use the client to fetch property data, trigger enrichments, or invoke an agent workflow.

example.ts 
import { Windfield } from '@windfield/sdk';

// Initialize the client
const wf = new Windfield({
  apiKey: process.env.WINDFIELD_API_KEY,
  baseUrl: 'https://api.windfield.io/v1'
});

// Fetch a property profile
const property = await wf.properties.get('prop_8x7k2m');
console.log(property.name, property.score);

// Trigger an enrichment workflow
await wf.agents.trigger('intelligence-agent', {
  propertyId: property.id,
  tools: ['market-scan', 'comp-analysis']
});
